# <a name="get-started-with-the-mailchimp-connector"></a>MailChimp 커넥터 시작

이제 논리 앱을 만들어 시작할 수 있습니다. [논리 앱 만들기](../logic-apps/quickstart-create-first-logic-app-workflow.md)를 참조하세요.
