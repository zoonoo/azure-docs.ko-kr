# <a name="log-analytics-view-designer-tile-reference"></a>Log Analytics 뷰 디자이너 타일 참조
Log Analytics에서 뷰 디자이너를 사용하면 OMS 리포지토리에 있는 데이터의 서로 다른 시각화 요소를 포함하고 있는 OMS 콘솔에서 사용자 지정 보기를 만들 수 있습니다. 이 문서에서는 사용자 지정 보기에 사용할 수 있는 타일 각각의 설정에 대한 참조가 제공됩니다.

뷰 디자이너에 적용할 수 있는 다른 문서는 다음과 같습니다.

* [뷰 디자이너](log-analytics-view-designer.md) - 뷰 디자이너 개요 및 사용자 지정 보기를 만들고 편집하는 절차
* [시각화 요소 참조](log-analytics-view-designer-parts.md) - 사용자 지정 보기에 사용할 수 있는 타일 각각의 설정에 대한 참조

>[!NOTE]
> 작업 영역을 [새 Log Analytics 쿼리 언어](log-analytics-log-search-upgrade.md)로 업그레이드한 경우 모든 뷰의 쿼리를 [새 쿼리 언어](https://go.microsoft.com/fwlink/?linkid=856078)로 써야 합니다.  작업 영역을 업그레이드하기 전에 생성된 모든 뷰는 자동으로 변환됩니다.

다음 표에서는 뷰 디자이너에서 사용할 수 있는 다양한 유형의 타일을 나열합니다.  그 아래에 나오는 섹션에서는 각 타일 형식 및 해당 속성에 대해 자세히 설명합니다.

| 타일 | 설명 |
|:--- |:--- |
| [Number](#number-tile) |한 쿼리의 레코드 수를 보여 주는 1개의 단일 숫자 |
| [2개 숫자](#two-numbers-tile) |서로 다른 두 쿼리의 레코드 수를 보여 주는 2개의 단일 숫자입니다. |
| [도넛](#donut-tile) |중앙에 요약 값이 있는 쿼리 기반 도넛형 차트입니다. |
| [꺾은선형 차트 및 설명선](#line-chart-amp-callout-tile) |쿼리 기반 꺾은선형 차트 및 요약 값이 있는 설명선입니다. |
| [꺾은선형 차트](#line-chart-tile) |쿼리 기반 꺾은선형 차트입니다. |
| [2개 타임 라인](#two-timelines-tile) |각각 별도의 쿼리에 기반하는 두 계열이 있는 세로 막대형 차트입니다. |

## <a name="number-tile"></a>1개 숫자 타일
**1개 숫자** 타일에서는 로그 쿼리의 레코드 수를 나타내는 하나의 단일 숫자와 레이블이 표시됩니다.

![1개 숫자 타일](media/log-analytics-view-designer/tile-number.png)

| 설정 | 설명 |
|:--- |:--- |
| 이름 |타일 위쪽에 표시할 텍스트입니다. |
| 설명 |타일 이름 아래에 표시할 텍스트입니다. |
| **타일** | |
| 범례 |값 아래에 표시할 텍스트입니다. |
| 쿼리 |실행할 쿼리입니다.  쿼리에서 반환하는 레코드 수가 표시됩니다. |
| **고급** |**> 데이터 흐름 확인** |
| 사용 |타일에 대해 데이터 흐름 확인을 사용해야 하는 경우에 선택합니다.  이렇게 하면 타일에 사용할 수 없는 데이터일 때 대체 메시지를 제공합니다.  일반적으로 보기가 설치되고 데이터가 사용될 수 있을 때 일시적으로 메시지를 제공하는 데 사용됩니다. |
| 쿼리 |보기에 사용할 수 있는 데이터인지 확인하기 위해 실행할 쿼리입니다.  쿼리에서 결과를 반환하지 않는 경우 기본 쿼리의 값 대신 메시지가 표시됩니다. |
| Message |데이터 흐름 확인 쿼리에서 데이터를 반환하지 않는 경우에 표시할 메시지입니다.  메시지를 제공하지 않을 경우 *평가 수행 중*이라는 메시지가 표시됩니다. |

## <a name="donut-tile"></a>도넛형 타일
**도넛형** 타일에서 로그 쿼리의 값 열로부터 요약된 숫자 하나를 표시합니다.  도넛에서 상위 3개 레코드의 결과를 그래픽으로 표시합니다.

![도넛형 타일](media/log-analytics-view-designer/tile-donut.png)

| 설정 | 설명 |
|:--- |:--- |
| 이름 |타일 위쪽에 표시할 텍스트입니다. |
| 설명 |타일 이름 아래에 표시할 텍스트입니다. |
| **도넛** | |
| 쿼리 |도넛을 위해 실행할 쿼리입니다.  첫 번째 속성은 텍스트 값, 두 번째 속성은 숫자 값이어야 합니다.  이 쿼리는 일반적으로 **measure** 키워드를 사용하여 결과를 요약하는 쿼리입니다. |
| **도넛** |**> 중앙** |
| 텍스트 |도넛의 값 밑에 표시할 텍스트입니다. |
| 작업 |값 속성에서 수행하여 단일 값으로 요약할 연산 작업입니다.<br><br>- 합계: 속성 값을 가진 모든 레코드의 값을 추가합니다.<br>- 백분율: 모든 레코드의 합계 값 중에서 속성 값을 가진 레코드의 합계 값이 차지하는 백분율입니다. |
| 중앙에 사용되는 결과 값 연산 |필요에 따라 +(더하기) 기호를 클릭하여 값을 하나 이상 추가합니다.  쿼리 결과는 지정한 속성 값을 포함하는 레코드로 제한됩니다.  값이 추가되지 않으면 쿼리에 모든 레코드가 포함됩니다. |
| **도넛** |**> 추가 옵션** |
| 색 |상위 3개 속성 각각에 대해 표시할 색입니다.  특정 속성 값의 대체 색을 지정하려면 고급 색 매핑을 사용합니다. |
| 고급 색 매핑 |특정 속성 값의 색을 표시합니다.  지정하는 값이 상위 3개 속성에 속하면 표준 색 대신 대체 색이 표시됩니다.  상위 3개에 속하지 않는 속성인 경우에는 색이 표시되지 않습니다. |
| **고급** |**> 데이터 흐름 확인** |
| 사용 |타일에 대해 데이터 흐름 확인을 사용해야 하는 경우에 선택합니다.  이렇게 하면 타일에 사용할 수 없는 데이터일 때 대체 메시지를 제공합니다.  일반적으로 보기가 설치되고 데이터가 사용될 수 있을 때 일시적으로 메시지를 제공하는 데 사용됩니다. |
| 쿼리 |보기에 사용할 수 있는 데이터인지 확인하기 위해 실행할 쿼리입니다.  쿼리에서 결과를 반환하지 않는 경우 기본 쿼리의 값 대신 메시지가 표시됩니다. |
| Message |데이터 흐름 확인 쿼리에서 데이터를 반환하지 않는 경우에 표시할 메시지입니다.  메시지를 제공하지 않을 경우 *평가 수행 중*이라는 메시지가 표시됩니다. |


## <a name="line-chart-tile"></a>꺾은선형 차트 타일
**꺾은선형 차트** 타일에서 로그 쿼리의 여러 시계열이 있는 꺾은선형 차트를 표시합니다.  

![꺾은선형 차트 및 설명선 타일](media/log-analytics-view-designer/tile-line-chart.png)

| 설정 | 설명 |
|:--- |:--- |
| 이름 |타일 위쪽에 표시할 텍스트입니다. |
| 설명 |타일 이름 아래에 표시할 텍스트입니다. |
| **꺾은선형 차트** | |
| 쿼리 |꺾은선형 차트를 위해 실행할 쿼리입니다.  첫 번째 속성은 텍스트 값, 두 번째 속성은 숫자 값이어야 합니다.  이 쿼리는 일반적으로 **measure** 키워드를 사용하여 결과를 요약하는 쿼리입니다.  쿼리에서 **interval** 키워드를 사용하는 경우 차트의 X축에 이 시간 간격이 사용됩니다.  쿼리에서 **interval** 키워드를 사용하지 않는 경우에는 X축에 1시간 간격이 사용됩니다. |
| **꺾은선형 차트** |**> Y축** |
| 로그 눈금 간격 사용 |선택하는 경우 Y축에 로그 눈금 간격을 사용합니다. |
| Units |쿼리에서 반환하는 값의 단위를 지정합니다.  값 형식을 나타내고 선택적으로 값을 변환하는 레이블을 차트에 표시하는 데 유용한 정보입니다.  **단위 형식**은 단위의 범주를 지정하고 사용 가능한 **현재 단위 형식** 값을 정의합니다.  **변환 대상**에서 값을 선택하는 경우 숫자 값이 **현재 단위** 형식에서 **변환 대상** 형식으로 변환됩니다. |
| 사용자 지정 레이블 |단위 형식의 레이블 옆에 있는 Y축에 표시할 텍스트입니다.  레이블이 지정되지 않으면 단위 형식만 표시합니다. |
| **고급** |**> 데이터 흐름 확인** |
| 사용 |타일에 대해 데이터 흐름 확인을 사용해야 하는 경우에 선택합니다.  이렇게 하면 타일에 사용할 수 없는 데이터일 때 대체 메시지를 제공합니다.  일반적으로 보기가 설치되고 데이터가 사용될 수 있을 때 일시적으로 메시지를 제공하는 데 사용됩니다. |
| 쿼리 |보기에 사용할 수 있는 데이터인지 확인하기 위해 실행할 쿼리입니다.  쿼리에서 결과를 반환하지 않는 경우 기본 쿼리의 값 대신 메시지가 표시됩니다. |
| Message |데이터 흐름 확인 쿼리에서 데이터를 반환하지 않는 경우에 표시할 메시지입니다.  메시지를 제공하지 않을 경우 *평가 수행 중*이라는 메시지가 표시됩니다. |


## <a name="line-chart--callout-tile"></a>꺾은선형 차트 및 설명선 타일
**꺾은선형 차트 및 설명선** 타일에서 로그 쿼리의 여러 시계열 및 요약된 값이 있는 설명선을 포함한 꺾은선형 차트를 표시합니다.  

![꺾은선형 차트 및 설명선 타일](media/log-analytics-view-designer/tile-line-chart-callout.png)

| 설정 | 설명 |
|:--- |:--- |
| 이름 |타일 위쪽에 표시할 텍스트입니다. |
| 설명 |타일 이름 아래에 표시할 텍스트입니다. |
| **꺾은선형 차트** | |
| 쿼리 |꺾은선형 차트를 위해 실행할 쿼리입니다.  첫 번째 속성은 텍스트 값, 두 번째 속성은 숫자 값이어야 합니다.  이 쿼리는 일반적으로 **measure** 키워드를 사용하여 결과를 요약하는 쿼리입니다.  쿼리에서 **interval** 키워드를 사용하는 경우 차트의 X축에 이 시간 간격이 사용됩니다.  쿼리에서 **interval** 키워드를 사용하지 않는 경우에는 X축에 1시간 간격이 사용됩니다. |
| **꺾은선형 차트** |**> 설명선** |
| 설명선 제목 |설명선 값 위에 표시할 텍스트입니다. |
| 계열 이름 |설명선 값에 사용할 계열의 속성 값입니다.  계열이 제공되지 않으면 쿼리에서 모든 레코드가 사용됩니다. |
| 작업 |값 속성에서 수행하여 설명선의 단일 값으로 요약할 연산 작업입니다.<br>- 평균: 모든 레코드 값의 평균입니다.<br><br>- 개수: 쿼리에서 반환하는 전체 레코드의 수입니다.<br>- 최종 샘플: 차트에 포함된 마지막 간격의 값입니다.<br>- 최대: 차트에 포함된 간격의 최대 값입니다.<br>- 최소: 차트에 포함된 간격의 최소 값입니다.<br>- 합계: 모든 레코드 값의 합계입니다. |
| **꺾은선형 차트** |**> Y축** |
| 로그 눈금 간격 사용 |선택하는 경우 Y축에 로그 눈금 간격을 사용합니다. |
| Units |쿼리에서 반환하는 값의 단위를 지정합니다.  값 형식을 나타내고 선택적으로 값을 변환하는 레이블을 차트에 표시하는 데 유용한 정보입니다.  **단위 형식**은 단위의 범주를 지정하고 사용 가능한 **현재 단위 형식** 값을 정의합니다.  **변환 대상**에서 값을 선택하는 경우 숫자 값이 **현재 단위** 형식에서 **변환 대상** 형식으로 변환됩니다. |
| 사용자 지정 레이블 |단위 형식의 레이블 옆에 있는 Y축에 표시할 텍스트입니다.  레이블이 지정되지 않으면 단위 형식만 표시합니다. |

| 사용 |타일에 대해 데이터 흐름 확인을 사용해야 하는 경우에 선택합니다.  이렇게 하면 타일에 사용할 수 없는 데이터일 때 대체 메시지를 제공합니다.  일반적으로 보기가 설치되고 데이터가 사용될 수 있을 때 일시적으로 메시지를 제공하는 데 사용됩니다. |
| 쿼리 |보기에 사용할 수 있는 데이터인지 확인하기 위해 실행할 쿼리입니다.  쿼리에서 결과를 반환하지 않는 경우 기본 쿼리의 값 대신 메시지가 표시됩니다. |
| Message |데이터 흐름 확인 쿼리에서 데이터를 반환하지 않는 경우에 표시할 메시지입니다.  메시지를 제공하지 않을 경우 *평가 수행 중*이라는 메시지가 표시됩니다. |
