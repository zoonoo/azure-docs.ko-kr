# <a name="connect-one-identity-safeguard"></a>One Identity Safeguard 연결
One Identity Safeguard 데이터 커넥터는 Safeguard for Privileged 세션 관련 대시보드의 보호를 사용하여 표준 CEF(Common Event Format) 커넥터를 향상합니다. 해당 커넥터를 사용하여 시각화, 경고, 조사 등에 대해 디바이스에서 생성한 이벤트를 간편하게 사용하기 시작할 수 있습니다.


## <a name="prerequisites"></a>사전 요구 사항

- Azure Sentinel 작업 영역에 대한 읽기 및 쓰기 권한.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>One Identity Safeguard에 Azure Sentinel 연결

1. Azure Portal에서 **Azure Sentinel** > **데이터 커넥터** 로 이동한 다음 **One Identity Safeguard** 커넥터를 선택합니다.

2. **커넥터 페이지 열기** 를 선택합니다.

3. **One Identity Safeguard** 페이지에 있는 지침을 따르세요.
