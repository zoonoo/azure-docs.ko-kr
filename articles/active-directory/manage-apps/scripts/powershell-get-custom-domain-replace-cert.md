# <a name="get-all-application-proxy-applications-published-with-the-identical-certificate-and-replace-it"></a>동일한 인증서를 사용하여 게시된 모든 애플리케이션 프록시 애플리케이션을 가져와서 바꾸기

이 PowerShell 스크립트 예제를 사용하면 동일한 인증서를 사용하여 게시된 모든 Azure AD(Azure Active Directory) 애플리케이션 프록시 애플리케이션의 인증서를 일괄적으로 바꿀 수 있습니다.

## <a name="script-explanation"></a>스크립트 설명

| 명령 | 메모 |
|---|---|
|[Get-AzureADServicePrincipal](/powershell/module/azuread/get-azureadserviceprincipal?view=azureadps-2.0) | 서비스 주체를 가져옵니다. |
|[Get-AzureADApplication](/powershell/module/azuread/get-azureadapplication?view=azureadps-2.0) | Azure AD 애플리케이션을 가져옵니다. |
|[Get-AzureADApplicationProxyApplication](/powershell/module/azuread/get-azureadapplicationproxyapplication?view=azureadps-2.0) | Azure AD에서 애플리케이션 프록시에 대해 구성된 애플리케이션을 검색합니다. |
|[Set-AzureADApplicationProxyApplicationCustomDomainCertificate](/powershell/module/azuread/set-azureadapplicationproxyapplicationcustomdomaincertificate?view=azureadps-2.0) | Azure AD에서 애플리케이션 프록시에 대해 구성된 애플리케이션에 인증서를 할당합니다. 이 명령은 인증서를 업로드하고 애플리케이션에서 사용자 지정 도메인을 사용할 수 있도록 합니다. |
