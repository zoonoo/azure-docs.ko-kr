# <a name="view-your-privacy-related-info-from-the-my-profile-preview-portal"></a>내 프로필(미리 보기) 포털에서 개인 관련 정보 보기
**내 프로필** 포털의 **개인 정보** 페이지에서 조직의 데이터 사용 방법을 볼 수 있습니다.

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

## <a name="view-your-privacy-related-info"></a>개인 관련 정보 보기
1. 회사 또는 학교 계정에 로그인한 다음, https://myprofile.microsoft.com/ 페이지로 이동합니다.

2. 왼쪽 탐색 창에서 **개인 정보**를 선택하거나 **개인 정보** 블록에서 **개인 정보 검토** 링크를 선택합니다.

    ![강조 표시된 개인 정보 링크를 보여주는 내 프로필 페이지](media/myprofile-portal/myprofile-portal-privacy-links.png)

3. 다음을 포함한 정보를 검토합니다.

    - **서비스.** 회사 또는 학교 계정을 사용하여 연결된 온라인 서비스 목록입니다.

    - **사용 약관**. 조직의 사용 약관입니다.

## <a name="next-steps"></a>다음 단계

- 최근 로그인 활동을 확인한 후 [보안 정보](user-help-security-info-overview.md), [최근 로그인 활동](myprofile-portal-recent-activity-page.md), 연결된 [디바이스](myprofile-portal-devices-page.md) 또는 [ 조직](myprofile-portal-organizations-page.md)을 보거나 관리하도록 선택할 수 있습니다.
