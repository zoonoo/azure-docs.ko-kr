# <a name="azure-mobile-engagement-web-sdk-release-notes"></a>Azure Mobile Engagement 웹 SDK 릴리스 정보
> [!IMPORTANT]
> Azure Mobile Engagement는 2018/3/31에 사용이 중지됩니다. 이 페이지는 이후에 삭제됩니다.
> 

## <a name="202-10182016"></a>2.0.2 (10/18/2016)
* 개인 브라우징(Safari)의 작동 중단 문제가 해결되었습니다.
* 쿠키를 사용하지 않는 브라우저의 작동 중단 문제가 해결되었습니다.

## <a name="201-6102016"></a>2.0.1(6/10/2016)
* Internet Explorer 8 및 Internet Explorer 9에서 SDK가 사용하지 않도록 설정되었습니다.
* Opera 웹 브라우저 감지 문제가 수정되었습니다.

## <a name="200-672016"></a>2.0.0 (6/7/2016)
* Azure Mobile Engagement가 릴리스되었습니다.
* `appId` 구성이 연결 문자열 구성으로 바뀌도록 업데이트되었습니다.
* 에이전트를 활성화 및 비활성화하는 API가 추가되었습니다.
* 보안이 개선되었습니다.
* 브라우저에서 네이티브 JSON API를 사용하도록 업데이트되었습니다.
* 장치 간에 메시지를 보내고 받는 API가 제거되었습니다.
* 사용이 중단된 XMPP 연결과 관련된 콜백이 제거되었습니다.
* 도달률 기능에 대한 지원이 제거되었습니다.
