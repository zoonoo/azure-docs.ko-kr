# <a name="frequently-asked-questions-about-the-education-hub"></a>Education Hub에 대한 질문과 대답

이 문서에서는 Azure for Students, Azure for Students Starter 및 Azure Dev Tools for Teaching에 대한 질문과 대답을 다룹니다.

## <a name="azure-for-students"></a>Azure for Students

### <a name="what-happens-after-i-use-my-100-credit-or-im-at-the-end-of-12-months"></a>100달러 크레딧을 사용하거나 12개월이 지나면 어떻게 되나요?

사용 가능한 크레딧이 소진되거나 12개월이 지나면 Azure 구독을 사용할 수 없게 됩니다. [Azure](https://account.azure.com/)를 계속 사용하려면 Azure 계정 포털에서 종량제 구독으로 업그레이드할 수 있습니다. Azure for Students에 대한 자세한 사용 약관을 보려면 [제안 사용 약관](https://azure.microsoft.com/offers/ms-azr-0170p/)을 방문하세요.

### <a name="can-i-get-azure-for-students-again-next-year"></a>내년에 Azure for Students를 다시 구할 수 있나요?

Azure for Students에 관심을 가져 주셔서 감사드립니다. 만족스럽게 사용하신 것을 다행으로 생각합니다. 이번에는 갱신을 발표하지 않습니다. 지금 업데이트를 다시 확인하세요.

### <a name="what-are-subscriptions-and-how-do-they-relate-to-azure-for-students"></a>구독이란 무엇이며, Azure for Students와 어떤 관련이 있나요?

구독은 Azure 서비스에 대한 액세스를 제공합니다. Azure for Students는 12개월 동안 100달러의 크레딧을 제공합니다. 이 경우 컴퓨팅, 네트워크, 스토리지 및 데이터베이스를 포함하여 25개가 넘는 체험 서비스에 액세스할 수 있습니다. 이 기간 동안 발생하는 모든 요금은 크레딧에서 공제됩니다. 12개월이 지나거나 100달러 크레딧이 소진되는 경우 중 어느 것이 먼저 발생하든지 간에 Azure 서비스를 계속 사용하려면 종량제 구독으로 업그레이드해야 합니다.

### <a name="what-happens-with-my-azure-services-if-i-dont-upgrade"></a>업그레이드하지 않는 경우 Azure 서비스는 어떻게 되나요?

12개월이 지나거나 100달러 크레딧이 소진되는 경우 중 어느 것이 먼저 발생하든지 간에 업그레이드하지 않도록 결정하면 배포한 모든 제품의 서비스가 해제되어 해당 제품에 액세스할 수 없습니다. 체험 구독이 종료된 후 90일 이내에 종량제 구독으로 업그레이드할 수 있습니다.

### <a name="how-do-i-know-how-much-of-the-100-credit-i-have-left"></a>100달러 크레딧 중 남아 있는 크레딧을 확인하려면 어떻게 해야 하나요?

남아 있는 크레딧은 [Azure Sponsorships 포털](https://www.microsoftazuresponsorships.com/)에서 확인할 수 있습니다.

### <a name="how-do-i-download-the-software-developer-tools"></a>소프트웨어 개발자 도구를 다운로드하려면 어떻게 해야 하나요?

Azure for Students 구독을 통해 특정 소프트웨어 개발자 도구에 액세스할 수 있습니다. 소프트웨어 개발자 도구에 액세스하여 다운로드하려면 현재 활성 Azure for Students 구독이 있어야 합니다. Azure for Students 구독을 사용하여 소프트웨어 개발자 도구를 다운로드하려면 [Education Hub](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)로 이동합니다.

### <a name="what-is-microsoft-learn"></a>Microsoft Learn이란?

[Microsoft Learn](/learn/)은 Azure 기술을 원하는 속도로 학습할 수 있는 체험 온라인 학습 플랫폼입니다. 학습 경로에는 기본 방법으로 시작한 다음, 실제 과제를 처리하는 고급 방법으로 이동할 수 있는 모듈이 결합되어 있습니다.

### <a name="can-azure-for-students-be-used-for-production-or-only-for-development"></a>Azure for Students를 프로덕션 또는 개발 용도로만 사용할 수 있나요?

Azure for Students를 사용하면 교육, 강의, 비상업용 연구 또는 이러한 용도에 적합한 소프트웨어 애플리케이션의 설계, 개발, 테스트 및 시연을 특별히 지원하기 위한 모든 Azure 제품에 액세스할 수 있습니다.

### <a name="can-i-apply-any-of-my-100-credit-toward-azure-marketplace-offers"></a>내 100달러 크레딧을 Azure Marketplace 제안에 적용할 수 있나요?

아니요. Azure Marketplace 제안에는 크레딧을 적용할 수 없습니다. 그러나 많은 Azure Marketplace 파트너는 자체 솔루션에 대한 평가판 또는 체험 계층 계획을 제공합니다.

## <a name="azure-for-students-starter"></a>Azure for Students Starter

### <a name="what-is-azure-for-students-starter"></a>Azure for Students Starter란?

Azure for Students Starter를 사용하면 클라우드에서 개발하는 데 필요한 Azure 제품을 시작할 수 있습니다. 비용이 없습니다. 이 혜택을 통해 다음 서비스의 체험 계층에 액세스할 수 있습니다.

- Azure App Service
- Azure 기능
- Azure Notification Hubs
- Azure Database for MySQL
- Application Insights
- Azure DevOps Services(이전의 Visual Studio Team Services)

Azure for Students Starter는 비용을 들이지 않고 약정 또는 시간 제한 없이 확인된 학생에게 제공됩니다. 자세한 사용 약관은 [Azure for Students Starter 제안](https://azure.microsoft.com/offers/ms-azr-0144p/)을 참조하세요.

### <a name="who-is-eligible-for-azure-for-students-starter"></a>Azure for Students Starter에 적합한 사용자는 누구인가요?

Azure for Students Starter는 다음 요구 사항을 충족하는 학생에게만 제공됩니다.

- 미국에 거주하는 경우 13세 이상임을 확인해야 합니다.
- 미국 이외의 국가에 거주하는 경우 16세 이상임을 확인해야 합니다.
- 조직의 이메일 주소를 통해 학업 상태를 확인해야 합니다. 조직에서 지원하는 경우 Shibboleth를 사용할 수도 있습니다.

이 제안은 MOOC(대규모 공개 온라인 과정) 또는 영리 조직의 다른 전문 학습에서 사용할 수 없습니다.

이 제안은 적격 고객당 하나의 Azure for Students Starter 구독으로 제한됩니다. Microsoft에서 달리 허용하지 않으면 양도할 수 없으며 다른 제안과 결합할 수 없습니다.

### <a name="will-i-have-to-pay-something-at-any-point-in-time"></a>언제든지 요금을 지불해야 하나요?

Azure for Students Starter 제안에는 신용 카드가 필요하지 않습니다. 이 제안을 통해 제한된 Azure 서비스 세트에 액세스할 수 있습니다. 그러나 언제든지 종량제 구독으로 업그레이드하여 모든 Azure 서비스에 액세스할 수 있습니다.

### <a name="how-do-i-download-the-software-developer-tools"></a>소프트웨어 개발자 도구를 다운로드하려면 어떻게 해야 하나요?

Azure for Students 구독을 통해 무료로 다운로드할 수 있는 특정 소프트웨어 개발자 도구에 액세스할 수 있습니다.

소프트웨어 개발자 도구에 액세스하려면 현재 활성 Azure for Students 구독이 있어야 합니다.

이 소프트웨어는 [Education Hub](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)에서 다운로드할 수 있습니다.

### <a name="what-is-microsoft-learn"></a>Microsoft Learn이란?

[Microsoft Learn](/learn/)은 Azure 기술을 원하는 속도로 학습할 수 있는 체험 온라인 학습 플랫폼입니다. 학습 경로에는 기본 방법으로 시작한 다음, 실제 과제를 처리하는 고급 방법으로 이동할 수 있는 모듈이 결합되어 있습니다.

## <a name="azure-academic-grant"></a>Azure Academic Grant

### <a name="how-do-i-start-using-my-azure-course-credits"></a>내 Azure 과정 크레딧을 사용하려면 어떻게 해야 하나요?

새 Microsoft Azure Academic Grant 구독을 만들어 Azure 과정 크레딧에 액세스할 수 있습니다. 스폰서쉽 승인 이메일에서 **활성화** 단추를 선택합니다.

또한 기존 구독을 Microsoft Azure 스폰서쉽 제안으로 변환하여 크레딧에 액세스할 수도 있습니다. 구독을 변환하는 방법에 대한 자세한 내용은 다음 질문을 참조하세요.

### <a name="can-i-associate-my-course-credits-with-an-existing-subscription"></a>내 과정 크레딧을 기존 구독과 연결할 수 있나요?

과정 크레딧은 제안에 부여된 계정의 기존 구독과 연결할 수 있습니다. [Azure 지원](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview)에 문의하여 과정 크레딧을 연결하세요.

### <a name="why-do-i-see-a-0-balance-in-the-azure-portal"></a>Azure Portal에서 0달러 잔액으로 표시되는 이유는 무엇인가요?

Azure Portal에서 구독 세부 정보로 이동하면 0.00달러가 표시됩니다. 제안은 모든 서비스에 100% 할인을 제공합니다. 따라서 포털에는 매월 사용 기간 동안 청구되는 금액이 표시되며, 이로 인해 0.00달러가 표시됩니다.

잔액 및 스폰서쉽 정보를 확인하려면 [Azure Sponsorships](https://www.microsoftazuresponsorships.com/balance)로 이동하여 계정에 로그인합니다.

### <a name="can-i-apply-my-course-credits-to-an-existing-enterprise-agreement-ea"></a>내 과정 크레딧을 기존 EA(기업계약)에 적용할 수 있나요?

과정 크레딧 제안은 EA(기업계약)의 계정에 있는 구독에 연결할 수 없습니다.

과정 크레딧을 적용하려면 EA 외부에 있는 새 계정을 만들어야 합니다. 그러면 자격을 부여할 수 있습니다.

스폰서쉽 기간이 종료되면 해당 구독을 EA에 다시 연결할 수 있습니다.

> [!WARNING]
> 스폰서쉽이 종료되기 전에 계정을 EA에 연결하면 모든 스폰서쉽 적립 금액이 종료됩니다. 자세한 내용은 [Azure 스폰서쉽 제안](https://azure.microsoft.com/offers/ms-azr-0143p/)의 사용 약관을 참조하세요.

### <a name="can-i-apply-my-course-credits-to-my-csp-subscription"></a>내 과정 크레딧을 내 CSP 구독에 적용할 수 있나요?

아니요, 과정 크레딧 제안은 CSP(클라우드 솔루션 공급자) 구독과 연결할 수 없습니다.

### <a name="can-i-move-my-course-credits-to-another-account"></a>내 과정 크레딧을 다른 계정으로 이동할 수 있나요?

예, 과정 크레딧 자격은 다른 계정으로 이동할 수 있습니다. 이렇게 하려면 [Azure 지원](https://ms.portal.azure.com/#blade/Microsoft_Azure_Support/HelpAndSupportBlade/overview)에 문의하세요.

### <a name="are-azure-marketplace-applications-covered-by-course-credits"></a>Azure Marketplace 애플리케이션에는 과정 크레딧이 적용되나요?

아니요, 타사 애플리케이션에는 과정 크레딧이 적용되지 않습니다. 계정의 신용 카드로 청구됩니다. 자세한 내용은 다음을 참조하세요.

- [Azure Marketplace](https://azuremarketplace.microsoft.com/marketplace/)에서 마켓플레이스 서비스를 확인합니다.
- [Azure 스폰서쉽 제안](https://azure.microsoft.com/offers/ms-azr-0143p/) 사용 약관을 참조합니다.

### <a name="i-have-a-previous-balance-due-can-i-pay-it-off-with-my-course-credits"></a>이전 미지불액이 남아 있습니다. 내 과정 크레딧을 사용하여 이를 변제할 수 있나요?

Azure 과정 크레딧은 Azure 스폰서쉽 제안을 활성화한 시점부터의 사용에만 적용됩니다. 사용자는 제안 시작 날짜 이전의 모든 요금을 지불해야 합니다.

### <a name="how-do-i-know-if-my-subscription-is-on-the-azure-course-credit-offer"></a>내 구독이 Azure 과정 크레딧 제안에 있는지 여부를 어떻게 알 수 있나요?

Azure Portal의 [구독] 블레이드에서 특정 구독을 살펴보면 **제안 이름** 이 속성 중 하나로 표시됩니다. [제안 이름]이 과정 크레딧에 연결된 경우 **Azure 스폰서쉽** 으로 표시됩니다. **Azure 스폰서쉽** 이 표시되지 않으면 지원에 문의하여 변환하세요.

### <a name="how-do-i-download-software"></a>소프트웨어를 다운로드하려면 어떻게 해야 하나요?

Microsoft Azure Dev Tools for Teaching 구독을 통해 특정 소프트웨어 개발자 도구에 액세스할 수 있습니다. 이러한 도구는 무료로 다운로드할 수 있습니다.

이 소프트웨어는 [Education Hub 소프트웨어 섹션](https://ms.portal.azure.com/#blade/Microsoft_Azure_Education/EducationMenuBlade/software)에서 다운로드할 수 있습니다.

### <a name="how-do-we-distribute-software-to-our-students"></a>학생들에게 소프트웨어를 배포하려면 어떻게 해야 하나요?

Azure Dev Tools for Teaching 구독자인 학교 또는 교육 기관은 Education Hub Store에 액세스할 수 있습니다. 학생들은 [Azure Portal](https://portal.azure.com/)에 있는 [Education Hub Store](https://azureforeducation.microsoft.com/devtools)에서 클라우드 서비스에 액세스합니다.

학생들은 학교(또는 Azure Dev Tools for Teaching) 자격 증명을 사용하여 Azure Portal에 로그인합니다. 그런 다음, 학생들은 Education Hub Store를 열고 사용 가능한 소프트웨어 다운로드에 액세스합니다.

### <a name="is-azure-dev-tools-for-teaching-available-internationally"></a>Azure Dev Tool for Teaching은 국제적으로 사용할 수 있나요?

예, Azure를 상업적으로 사용할 수 있는 140개가 넘는 국가/지역에서 사용할 수 있습니다.

### <a name="which-languages-are-available-in-the-software-to-end-users"></a>소프트웨어에서 최종 사용자가 사용할 수 있는 언어는 무엇인가요?

Education Hub Store는 아랍어, 중국어 간체, 중국어 번체, 덴마크어, 네덜란드어, 영어, 프랑스어, 독일어, 히브리어, 이탈리아어, 일본어, 한국어, 포르투갈어, 러시아어, 스페인어, 스웨덴어 및 터키어로 제공됩니다.

### <a name="if-our-students-download-software-through-the-azure-dev-tools-for-teaching-program-do-they-get-unlimited-use-of-the-software"></a>학생들이 Azure Dev Tools for Teaching 프로그램을 통해 소프트웨어를 다운로드하는 경우 해당 소프트웨어를 무제한으로 사용할 수 있나요?

예. 학생들은 학습 및 연구 노력을 향상시키기 위해 소프트웨어를 무제한으로 사용할 수 있습니다.

### <a name="if-our-students-create-viable-apps-and-products-using-azure-dev-tools-for-teaching-software-can-they-sell-them-commercially"></a>학생들이 Azure Dev Tools for Teaching 소프트웨어를 사용하여 실행 가능한 앱과 제품을 만드는 경우 상업적으로 판매할 수 있나요?

일반적으로 그렇지 않습니다. 학생들은 Azure Dev Tools for Teaching을 사용하여 만든 앱과 제품을 판매할 수 없습니다. 그러나 Azure Dev Tools for Teaching 및 Microsoft App Store 팀 간의 파트너 관계 덕분에 학생들은 Windows 스토어에서 판매할 게임과 애플리케이션을 만들 수 있습니다.

### <a name="do-i-have-unlimited-use-of-the-software-through-the-azure-dev-tools-for-teaching-program"></a>Azure Dev Tools for Teaching 프로그램을 통해 소프트웨어를 무제한으로 사용할 수 있나요?

예. 승인된 과정에 등록된 교직원은 상업용 이외의 용도로 사용할 수 있도록 Azure Dev Tools for Teaching 소프트웨어를 개인 컴퓨터에 설치할 수 있습니다.

### <a name="how-do-i-access-my-visual-studio-enterprise-benefit"></a>내 Visual Studio Enterprise 혜택에 액세스하려면 어떻게 해야 하나요?

Azure Dev Tools for Teaching 구독의 관리자는 [Azure Dev Tools for Teaching 관리 포털](https://azureforeducation.microsoft.com/account/Subscriptions)을 통해 액세스를 요청하여 Visual Studio Enterprise 구독에 액세스할 수 있습니다.

승인되면 [Visual Studio 포털](https://my.visualstudio.com/)에 로그인하여 추가 혜택을 받을 수 있습니다.

### <a name="does-microsoft-azure-dev-tools-for-teaching-include-microsoft-office"></a>Microsoft Azure Dev Tools for Teaching에 Microsoft Office가 포함되어 있나요?

아니요. Microsoft Azure Dev Tools for Teaching은 학부, 교직원 및 학생들에게 소프트웨어 개발 및 테스트에 대한 연구를 구체적으로 확장하는 데 필요한 도구를 제공하는 데 중점을 둡니다. 따라서 Windows Server, Visual Studio .NET, SQL Server 및 Platform SDK와 같은 기술이 제공됩니다.

### <a name="does-azure-dev-tools-for-teaching-include-azure-credit"></a>Azure Dev Tools for Teaching에 Azure 크레딧이 포함되나요?

아니요, Microsoft Azure Dev Tools for Teaching 구독에는 Azure 크레딧이 포함되지 않습니다. 그러나 Azure for Students에 가입할 수 있습니다. 이 경우 Azure 서비스에 대한 비용을 지불하는 데 사용할 100달러 상당의 Azure 크레딧이 제공됩니다. 자세한 내용은 [Azure for Students로 미래에 대비하세요!](https://aka.ms/student)로 이동하세요.

### <a name="do-students-need-an-office-365-or-active-directory-account-to-access-azure-dev-tools-for-teaching"></a>학생들이 Azure Dev Tools for Teaching에 액세스하려면 Office 365 또는 Active Directory 계정이 필요한가요?

아니요. 학생들은 Office 365 계정이 필요하지 않습니다. 학생들이 Active Directory 계정에 액세스할 수 있는 경우 동일한 자격 증명을 사용하여 소프트웨어에 로그인합니다.

학생들이 Active Directory를 사용하지 않는 경우 제공되는 것과 동일한 이메일 주소를 사용하여 [Microsoft 계정](https://account.microsoft.com/account)을 만들어야 합니다(아직 없는 경우).

### <a name="why-arent-my-sign-in-credentials-recognized-when-i-sign-in-to-azure-dev-tools-for-teaching"></a>Azure Dev Tools for Teaching에 로그인할 때 내 로그인 자격 증명이 인식되지 않는 이유는 무엇인가요?

학교 자격 증명을 사용하여 Azure Dev Tools for Teaching에 로그인하고 있는지 확인합니다. 프라이빗 검색 창 세션을 열면 도움이 될 수 있습니다.

여전히 로그인할 수 없는 경우 구독 관리자에게 문의하세요. 구독 관리자를 찾으려면 [지원에 문의](https://aka.ms/adt4tsupport)하세요.

### <a name="how-do-i-find-my-subscriber-id"></a>내 구독자 ID를 확인하려면 어떻게 해야 하나요?

- **프로그램에 처음 등록하는 경우** : 구독자 ID 번호는 받은 구독 환영 이메일에 있습니다.
- **구독을 갱신한 경우** : 구독자 ID는 구독 관리자가 받은 갱신 이메일에 있습니다.

또한 구독자 ID는 Visual Studio 구독 포털에도 있습니다. 로그인한 후 **내 계정** 페이지의 **내 구독** 아래에서 확인합니다.

구독자 ID를 찾는 데 도움이 필요한 경우 [지원에 문의](https://azureforeducation.microsoft.com/institutions/Contact)하세요.

### <a name="are-we-automatically-enrolled-in-azure-dev-tools-for-teaching-if-we-receive-it-as-part-of-our-academic-volume-licensing-agreement"></a>교육용 볼륨 라이선싱 계약의 일부로 받는 경우 Azure Dev Tools for Teaching에 자동으로 등록되나요?

아니요, 교육용 볼륨 라이선싱 계약이 있으면 Microsoft에서 자동으로 등록하지 않습니다. 교육용 볼륨 라이선싱 계약에 포함되는 항목은 다음과 같습니다.

- EES(교육 솔루션 등록)
- OVS-ES(교육 솔루션용 Open Value 정기가입 계약)
- 캠퍼스 계약
- 학교 계약

교육용 볼륨 라이선스에 대해 받은 구독 환영 이메일의 적절한 판촉 코드를 사용하여 Azure Dev Tools for Teaching에 등록해야 합니다.

구독이 만료되는 경우에도 해당 구독을 갱신해야 합니다. 이는 자동으로 갱신되지 않습니다.

판촉 코드를 찾을 수 없는 경우 [지원에 문의](https://azureforeducation.microsoft.com/institutions/Contact)하세요.

### <a name="how-and-when-do-we-renew-our-azure-dev-tools-for-teaching-subscription"></a>Azure Dev Tools for Teaching 구독을 갱신하려면 언제 어떻게 해야 하나요?

멤버 자격이 만료되기 60일 전에 구독을 갱신하도록 요청하는 이메일 미리 알림을 받기 시작합니다. 갱신 이메일에서 [갱신 링크](https://portal.azureforeducation.microsoft.com/)를 선택하세요.

이러한 미리 알림 이메일을 받지 못하여 구독이 곧 만료될 것으로 염려하는 경우 [지원에 문의](https://aka.ms/adt4tsupport)하세요.

또는 [Azure Dev Tools for Teaching - 갱신 페이지](https://portal.azureforeducation.microsoft.com/)로 이동하여 갱신 프로세스를 완료합니다.

구독 만료 날짜를 확인하려면 [Azure Dev Tools for Teaching 관리 포털](https://azureforeducation.microsoft.com/account/Subscriptions)로 이동하여 **구독** 아래에서 확인합니다.

### <a name="what-if-i-need-more-help"></a>도움이 더 필요한 경우 어떻게 해야 하나요?

[구독 지원] 페이지로 이동하여 해당 지역을 찾아서 [지원에 문의](https://azureforeducation.microsoft.com/institutions/Contact)하세요.
