# <a name="azure-fxt-edge-filer-specifications"></a>Azure FXT Edge Filer 사양

이 문서에서는 Azure FXT Edge Filer 하드웨어 노드의 하드웨어 사양을 설명합니다. 실제로 세 개 이상의 노드가 함께 클러스터 캐시 시스템을 제공하도록 구성됩니다.

## <a name="hardware-specifications"></a>하드웨어 사양

| 구성 요소 | FXT 6600 | FXT 6400 |
|----------|-----------|-----------|
| CPU 코어 |  16 | 16 |
| DRAM  | 1536 GB | 768 GB |
| 네트워크 포트 | 6 x 25/10Gb + 2 x 1Gb | 6 x 25/10Gb + 2 x 1Gb |
| NVMe SSD 용량 | 25.6TB | 12.8TB |

## <a name="drive-specifications"></a>드라이브 사양

시스템에는 전면에서 액세스할 수 있는 10개의 드라이브 베이가 있습니다. 채워진 각 드라이브의 오른쪽에는 용량 정보 레이블이 있습니다. 

드라이브 번호는 드라이브 사이의 공간에 인쇄됩니다. Azure FXT Edge Filer에서 드라이브 0은 왼쪽 위 드라이브이며 드라이브 1은 바로 아래에 있습니다.

![드라이브 번호 및 용량 레이블을 보여 주는 FXT 섀시에 있는 한 하드 드라이브 베이의 사진](media/fxt-drives-photo.png)

| 드라이브 번호    |  사용   |  사양 |
|------------------|--------|-----------------|
| 0, 1             | OS     | 480GB SATA SSD |
| 2, 3, 4, 5, 6, 7, 8, 9 | Data   | FXT 6600: 3.2TB NVMe SSD <br> FXT 6400: 1.6TB NVMe SSD |
