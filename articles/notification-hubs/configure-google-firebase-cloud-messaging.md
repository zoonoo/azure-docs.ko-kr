# <a name="configure-google-firebase-cloud-messaging-fcm-settings-for-a-notification-hub-in-the-azure-portal"></a>Azure portal에서 알림 허브에 대 한 Google Firebase Cloud Messaging (FCM) 설정 구성
이 문서에서는 Azure portal을 사용 하 여 Azure 알림 허브에 대 한 Google Firebase Cloud Messaging (FCM) 설정을 구성 하는 방법을 보여 줍니다.  

## <a name="prerequisites"></a>필수 조건
알림 허브를 아직 만들지 않은 경우 지금 만듭니다. 자세한 내용은 [Azure Portal에 Azure 알림 허브 만들기](create-notification-hub-portal.md)를 참조하세요. 

## <a name="configure-google-firebase-cloud-messaging-fcm"></a>Google Firebase Cloud Messaging (FCM) 구성

다음 절차는 알림 허브에 대 한 Google Firebase Cloud Messaging (FCM) 설정을 구성 하는 단계를 제공 합니다. 

1. Azure portal에서에 **알림 허브** 페이지에서 **Google (GCM/FCM)** 왼쪽된 메뉴에서. 
2. 앞에서 저장한 FCM 프로젝트의 **API 키**를 붙여넣습니다. 
3. **저장**을 선택합니다.
