# <a name="texconv---texture-conversion-tool"></a>TexConv-질감 변환 도구

TexConv 도구는 일괄 처리 된 질감 변환을 수행 하는 외부 명령줄 도구입니다.
이 도구는 git 리포지토리: [TexConv 도구용 GitHub 리포지토리](https://github.com/microsoft/DirectXTex/wiki/Texconv)에서 찾을 수 있습니다.

변환 단계에서 일반적인 이미지 파일 형식을 처리 하 고 질감을 적절 한 런타임 형식으로 변환할 수 있기 때문에 원격 렌더링의 경우 텍스처를 변환할 필요가 없습니다. 그러나이 도구는 질감을 사용자 지정 설정으로 로컬에서 변환 하는 데 유용 합니다.
