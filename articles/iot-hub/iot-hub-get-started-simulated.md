# <a name="azure-iot-hub-get-started-with-simulated-devices-tutorials"></a>시뮬레이션된 장치로 시작하는 Azure IoT Hub 자습서

이러한 자습서는 Azure IoT Hub 및 장치 SDK를 소개합니다. 이 자습서에서는 IoT Hub의 기능을 설명하기 위한 일반적인 IoT 시나리오를 다룹니다. 또한 IoT Hub를 다른 Azure 서비스 및 도구와 결합하여 좀 더 강력한 IoT 솔루션을 구축하는 방법도 보여 줍니다. 다음 표에 나열된 자습서는 시뮬레이션된 IoT 장치를 만드는 방법을 보여 줍니다.

| 프로그래밍 언어 |
|----------------------|
| [.NET][Sim_NET]      |
| [Java][Sim_Jav]      |
| [Node.js][Sim_Nd]    |
| [Python][Sim_Pyth]   |

또한 IoT Edge 게이트웨이를 사용하여 시뮬레이션된 장치를 IoT Hub에 연결할 수 있습니다.

| 프로그래밍 언어 | 플랫폼           |
|----------------------|------------------- |
| C                    | [Linux][Sim_Lnx]   |
| C                    | [Windows][Sim_Win] |
