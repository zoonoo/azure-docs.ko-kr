# <a name="access-a-vm-student-view-in-azure-lab-from-teams"></a>팀의 Azure 랩에서 VM (학생 보기)에 액세스

팀 내에서 랩을 만들 때 사용자는 팀 소유자가 프로 비전 한 모든 Vm을 보고 액세스할 수 있습니다. 랩을 게시 하 고 Vm을 만든 후에는 사용자가 자동으로 랩에 등록 되 고 Azure Lab Services에 대 한 첫 번째 로그인에서 VM이 할당 됩니다. 사용자는 **Azure Lab Services** 앱을 포함 하는 탭을 선택 하 여 할당 된 VM을 보고 액세스할 수 있습니다.

랩을 아직 게시 하지 않았거나 팀에 추가 된 후에도 동기화를 트리거하지 않은 경우 학생에 게 비슷한 메시지가 표시 됩니다.
