# <a name="resource-limits-for-single-databases-using-the-vcore-purchasing-model"></a>vCore 구매 모델을 사용한 단일 데이터베이스에 대한 리소스 제한
[!INCLUDE[appliesto-sqldb](../includes/appliesto-sqldb.md)]

이 문서에서는 vCore 구매 모델을 사용 하 여 Azure SQL Database의 단일 데이터베이스에 대 한 자세한 리소스 제한을 제공 합니다.

서버에 있는 단일 데이터베이스에 대 한 DTU 구매 모델 제한의 경우 [서버의 리소스 제한 개요](resource-limits-logical-server.md)를 참조 하세요.

[Azure Portal](single-database-manage.md#the-azure-portal), [transact-sql](single-database-manage.md#transact-sql-t-sql), [PowerShell](single-database-manage.md#powershell), [Azure CLI](single-database-manage.md#the-azure-cli)또는 [REST API](single-database-manage.md#rest-api)를 사용 하 여 단일 데이터베이스의 서비스 계층, 계산 크기 (서비스 목표) 및 저장소 크기를 설정할 수 있습니다.

> [!IMPORTANT]
> 크기 조정 지침 및 고려 사항은 [단일 데이터베이스 크기 조정](single-database-scale.md)을 참조 하세요.
