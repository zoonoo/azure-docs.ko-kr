# <a name="init-image-transformation"></a>이미지 변환 시작

이 문서에서는 Azure Machine Learning 디자이너 **에서 이미지 변환 초기화 모듈을** 사용 하 여 이미지를 변환 하는 방법을 지정 하는 이미지 변환을 초기화 하는 방법을 설명 합니다.

## <a name="how-to-configure-init-image-transformation"></a>초기화 이미지 변환을 구성 하는 방법

1.  디자이너에서 파이프라인에 **Init Image 변환** 모듈을 추가 합니다. 

2.  **크기 조정**의 경우 입력 Pil 이미지의 크기를 지정 된 크기로 조정할지 여부를 지정 합니다. ' T r u e '를 선택 하는 경우 원하는 출력 이미지 **크기를 기본적으로 256**으로 지정할 수 있습니다. 

3.  **가운데 자르기**의 경우 가운데에서 지정 된 Pil 이미지를 잘라낼 지 여부를 지정 합니다. ' T r u e '를 선택 하는 경우 자르기 **크기**에서 자르기의 원하는 출력 이미지 크기 (기본적으로 224)를 지정할 수 있습니다.  

4.  **Pad**의 경우 지정 된 Pil 이미지를 모든 면에 채움 값 0으로 채울 것인지 여부를 지정 합니다. ' T r u e '를 선택 하면 각 테두리 **에서 패딩 (** 더할 픽셀 수)을 지정할 수 있습니다.

5.  **색 지터**의 경우 이미지의 밝기, 대비 및 채도를 임의로 변경 하는지 여부를 지정 합니다.

6.  **회색조**의 경우 이미지를 회색조로 변환할지 여부를 지정 합니다.

7.  **크기 조정 된 임의 자르기**의 경우 지정 된 Pil 이미지를 임의 크기 및 가로 세로 비율로 잘라낼 지 여부를 지정 합니다. 원래 가로 세로 비율의 무작위 크기 (0.08에서 1.0 사이)와 임의 가로 세로 비율 (3/4에서 4/3 까지의 범위)에 대 한 자르기입니다. 마지막으로이 자르기의 크기가 지정 된 크기로 조정 됩니다.
    이는 일반적으로 개시 네트워크를 학습 하는 데 사용 됩니다. ' T r u e '를 선택 하는 경우에는 기본적으로 256에 따라 각 가장자리의 예상 출력 크기를 **임의로**지정할 수 있습니다.

8.  **임의 자르기**의 경우 지정 된 Pil 이미지를 임의 위치에서 잘라낼 지 여부를 지정 합니다. ' T r u e '를 선택 하는 경우 **임의 자르기 크기**에서 자르기의 원하는 출력 크기 (기본적으로 224)를 지정할 수 있습니다.

9.  **임의 수평 대칭 이동**의 경우 지정 된 Pil 이미지를 해당 확률 0.5로 무작위로 대칭 이동 하는지 여부를 지정 합니다.

10.  **임의 수직 대칭 이동**의 경우 지정 된 Pil 이미지를 확률 0.5로 무작위로 대칭 이동 하는지 여부를 지정 합니다.

11.  **임의 회전**의 경우 이미지를 각도로 회전할지 여부를 지정 합니다. ' T r u e '를 선택 하는 경우 **임의 회전 각도**를 설정 하 여 각도를 지정할 수 있습니다. 즉, 기본값은 0입니다.

12.  **임의 상관 관계**의 경우 중앙의 고정을 유지 하는 이미지의 임의 관계 변환을 수행할지 여부를 지정 합니다. ' T r u e '를 선택 하는 경우에는 각도의 범위 내에서 선택할 수 있습니다. **즉, (** 각도, +도)는 기본적으로 0입니다.

13.  **임의 회색조**의 경우 확률 0.1을 사용 하 여 이미지를 회색조로 무작위로 변환할지 여부를 지정 합니다.

14.  **임의 큐브 뷰에서**는 지정 된 Pil 이미지의 원근 변환을 확률 0.5로 무작위로 수행할지 여부를 지정 합니다.


16.  위에 지정 된 변환을 입력 이미지 데이터 집합에 적용 하려면 [이미지 변환 적용](apply-image-transformation.md) 모듈에 연결 합니다.

17. 파이프라인을 제출합니다.

## <a name="results"></a>결과

변환이 완료 된 후에 [이미지 변환 적용](apply-image-transformation.md) 모듈의 출력에서 변형 된 이미지를 찾을 수 있습니다.
