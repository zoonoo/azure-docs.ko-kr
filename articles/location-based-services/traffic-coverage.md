# <a name="azure-location-based-services---traffic-coverage"></a>Azure Location Based Services - 트래픽 검사

Azure Location Based Services(파운드)는 Azure LBS 맵 컨트롤의 맨 위에 오버레이로 처리되거나 Azure LBS Route Service와 함께 사용될 수 있는 흐름 및 인시던트 형식으로 풍부한 트래픽 정보를 제공합니다. 다음 표는 Azure LBS Traffic에 대한 검사 정보를 제공하여 사용자가 트래픽 정보를 예상할 수 있는(또는 예상할 수 없는) 위치에 대한 통찰을 할 수 있습니다.
