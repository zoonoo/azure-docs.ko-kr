# <a name="activity-management-overview"></a>활동 관리 개요

CloudSimple은 프라이빗 클라우드 환경의 기능에 영향을 줄 수 있는 모든 활동을 추적합니다. 활동에는 경고, 이벤트, 작업 및 감사 활동이 포함됩니다. [활동 페이지에서는](monitor-activity.md) 모든 현재 활동을 요약하고 추가 세부 정보를 드릴다운할 수 있습니다.

## <a name="events"></a>이벤트

이벤트는 CloudSimple 포털에서 사용자 및 시스템 활동을 추적합니다. 이벤트는 특정 리소스와 연관된 활동과 영향의 심각도를 표시합니다.  CloudSimple 포털에서 이벤트를 볼 수 있습니다.

## <a name="alerts"></a>경고

경고는 CloudSimple 환경에서 중요한 활동에 대한 알림입니다. 청구 또는 사용자 액세스에 영향을 주는 이벤트는 경고로 표시됩니다.  CloudSimple 포털에서 경고를 확인할 수 있습니다.

## <a name="tasks"></a>작업

작업을 완료하는 데 30초 이상 걸리는 모든 사용자 작업을 추적합니다. CloudSimple 포털에서 작업의 진행 률을 모니터링할 수 있습니다.  완료된 작업의 경우 정보에는 완료할 총 시간이 포함됩니다.

## <a name="audit"></a>감사

감사 로그는 사용자 작업을 추적합니다. 감사 로그에는 사용자가 작업에 대해 제공한 매개 변수가 포함됩니다.  감사 로그를 사용하여 모든 사용자의 사용자 활동을 모니터링할 수 있습니다.
