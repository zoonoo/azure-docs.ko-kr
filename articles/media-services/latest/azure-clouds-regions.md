# <a name="azure-media-services-v3-clouds-and-regions-availability"></a>Azure Media Services v3 클라우드 및 지역 가용성

[!INCLUDE [media services api v3 logo](./includes/v3-hr.md)]

Azure Media Services v3은 Azure Resource Manager를 통해 사용할 수 있습니다. 그러나 모든 Media Services 기능을 모든 Azure 클라우드에서 사용할 수 있는 것은 아닙니다. 이 문서에서는 기본 Media Services v3 구성 요소의 가용성을 설명합니다. 다음 표에서는 각 지역에서 사용할 수 있는 Media Services 기능을 보여 줍니다.
