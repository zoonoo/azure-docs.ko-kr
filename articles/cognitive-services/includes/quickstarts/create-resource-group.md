## <a name="create-a-resource-group"></a>리소스 그룹 만들기

Cognitive Services 리소스를 만들기 전에 계정에 리소스를 포함할 Azure 리소스 그룹이 있어야 합니다. 리소스 그룹이 아직 없는 경우 계속하기 전에 [Azure Portal](https://ms.portal.azure.com/)에서 만듭니다.
