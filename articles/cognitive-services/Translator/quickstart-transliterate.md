# <a name="quickstart-use-the-translator-to-transliterate-text"></a>빠른 시작: Translator를 사용하여 텍스트 음역

이 빠른 시작에서는 Translator REST API를 사용하여 텍스트를 한 스크립트에서 다른 스크립트로 음역(변환)하는 방법을 알아봅니다. 제공된 샘플에서는 라틴어 알파벳을 사용하도록 일본어를 음역합니다.

이 빠른 시작에는Translator 리소스가 있는 [Azure Cognitive Services 계정](https://docs.microsoft.com/azure/cognitive-services/cognitive-services-apis-create-account)이 필요합니다.
