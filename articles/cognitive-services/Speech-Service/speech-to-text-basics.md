# <a name="learn-the-basics-of-speech-recognition"></a>음성 인식의 기본 사항 알아보기

음성 서비스의 핵심 기능 중 하나는 사람의 음성을 인식하여 글로 바꾸는 기능입니다(종종 음성을 텍스트로 변환이라고도 함). 이 문서에서는 앱 및 제품에서 음성 SDK를 사용하여 고품질 음성 인식을 수행하는 방법을 알아봅니다.

> [!TIP]
> 아직 빠른 시작 중 하나를 완료하지 않은 분들은 음성 인식을 살펴보고 직접 경험해 보실 것을 권장합니다.
