이 가이드에서는 NodeJS와 함께 사용할 JavaScript용 [Speech SDK](~/articles/cognitive-services/speech-service/speech-sdk.md)를 설치하는 방법을 보여줍니다. 패키지 이름을 직접 시작하려면 기존 프로젝트에서 `npm install microsoft-cognitiveservices-speech-sdk`를 실행합니다.

[!INCLUDE [License Notice](~/includes/cognitive-services-speech-service-license-notice.md)]

## <a name="prerequisites"></a>사전 요구 사항

이 빠른 시작에는 다음이 필요합니다.

* [NodeJS](https://nodejs.org/)

## <a name="create-a-new-npm-project"></a>새 NPM 프로젝트를 만듭니다.

새 디렉터리를 만들고, `npm init`를 실행하고, 프롬프트를 안내합니다.

## <a name="add-the-speech-sdk-package"></a>Speech SDK 패키지 추가

`npm install microsoft-cognitiveservices-speech-sdk`을 실행합니다.
