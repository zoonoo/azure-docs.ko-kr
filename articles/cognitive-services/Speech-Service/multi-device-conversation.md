# <a name="what-is-multi-device-conversation-preview"></a>다중 장치 대화 (미리 보기) 란?

**다중 장치 대화** 를 사용 하면 여러 클라이언트 간에 음성 또는 텍스트 대화를 쉽게 만들고 두 클라이언트 간에 전송 되는 메시지를 조정할 수 있습니다.

**다중 장치 대화**를 사용 하 여 다음을 수행할 수 있습니다.

- 여러 클라이언트를 동일한 대화에 연결 하 고 두 클라이언트 간의 메시지 송수신을 관리 합니다.
- 각 클라이언트에서 오디오를 쉽게 높여줄 하 고 다른 번역을 사용 하 여 다른 사람에 게 기록을 보냅니다.
- 선택적 번역을 사용 하 여 클라이언트 간에 문자 메시지를 쉽게 보냅니다.

장치 배열에서 작동 하는 기능이 나 솔루션을 빌드할 수 있습니다. 각 장치는 다른 모든 장치에 메시지 (오디오 또는 인스턴트 메시지의 메시지)를 독립적으로 보낼 수 있습니다.

[**대화 기록을**](conversation-transcription.md) 다중 채널 마이크 배열을 사용 하는 단일 장치에서 작동 하는 반면 **다중 장치 대화** 는 여러 장치가 있는 시나리오에 적합 합니다. 각 장치에는 마이크가 하나 있습니다.

>[!IMPORTANT]
> 다중 장치 대화에서는 클라이언트 간에 오디오 파일 전송을 지원 **하지** 않습니다. 기록 및/또는 번역이 있습니다.

## <a name="key-features"></a>주요 기능

- **실시간 기록** -모든 사용자가 대화에 대 한 기록을 받게 되므로 실시간으로 텍스트를 따라 팔 로우 하거나 나중에 저장할 수 있습니다.
- **실시간 변환** – 텍스트 번역에 대해 [지원 되는 언어](language-support.md#text-languages) 를 60 개 이상 사용 하 여 사용자는 대화를 기본 설정 언어로 변환할 수 있습니다.
- **읽을 수** 있는 기록 – 문장 부호와 문장 나누기를 사용 하 여 기록과 번역을 쉽게 수행할 수 있습니다.
- **음성 또는 텍스트 입력** – 참가자가 선택한 언어에 대해 사용 하도록 설정 된 언어 지원 기능에 따라 각 사용자가 자신의 장치를 말하기 또는 입력할 수 있습니다. [언어 지원](language-support.md#speech-to-text)을 참조 하세요.
- **메시지 릴레이** -다중 장치 대화 서비스는 한 클라이언트에서 보낸 메시지를 선택한 언어로 모든 다른 클라이언트에 배포 합니다.
- **메시지 식별** – 대화에서 사용자에 게 표시 되는 모든 메시지에는 해당 메시지를 보낸 사용자의 애칭 태그가 지정 됩니다.

## <a name="use-cases"></a>사용 사례

### <a name="lightweight-conversations"></a>경량 대화

대화를 만들고 조인 하는 것은 쉽습니다. 한 명의 사용자가 ' 호스트 ' 역할을 하 고 대화를 만들어 임의의 5 자로 된 대화 코드 및 QR 코드를 생성 합니다. 다른 모든 사용자는 대화 코드를 입력 하거나 QR 코드를 스캔 하 여 대화에 참여할 수 있습니다. 

사용자는 대화 코드를 통해 연결 하 고 연락처 정보를 공유 하는 데 필요 하지 않으므로 신속 하 고 빠른 대화를 쉽게 만들 수 있습니다.

### <a name="inclusive-meetings"></a>포괄 모임

실시간 기록 및 번역은 다른 언어를 사용 하 고 있거나 청각 장애인 또는 청각 장애가 있는 사용자가 액세스할 수 있도록 하는 데 도움이 됩니다. 또한 각 사용자는 선호 하는 언어를 말하기 또는 인스턴트 메시지를 보내 대화에 적극적으로 참여할 수 있습니다.

### <a name="presentations"></a>프레젠테이션

화면과 대상 그룹 구성원의 장치에서 프레젠테이션 및 강의의 캡션을 제공할 수도 있습니다. 사용자가 대화 코드와 조인한 후에는 자신의 장치에서 선호 하는 언어로 기록을 볼 수 있습니다.

> [!NOTE]
> 예를 보려면 다중 장치 대화 서비스를 사용 하는 PowerPoint 추가 서비스인 [프레젠테이션 변환기](https://www.microsoft.com/translator/apps/presentation-translator/)를 확인 하세요. [여기](https://www.microsoft.com/download/details.aspx?id=55024)에서 다운로드할 수 있습니다.

## <a name="how-it-works"></a>작동 방법

모든 클라이언트는 음성 SDK를 사용 하 여 대화를 만들거나 조인 합니다. Speech SDK는 참가자 목록, 각 클라이언트의 선택 된 언어 및 전송 된 메시지를 포함 하 여 대화의 수명을 관리 하는 다중 장치 대화 서비스와 상호 작용 합니다.  

각 클라이언트는 오디오 또는 인스턴트 메시지를 보낼 수 있습니다. 이 서비스에서는 음성 인식을 사용 하 여 오디오를 텍스트로 변환 하 고 인스턴트 메시지를 그대로 보냅니다. 클라이언트에서 다른 언어를 선택 하는 경우 서비스는 모든 메시지를 각 클라이언트의 지정 된 언어로 변환 합니다.

## <a name="overview-of-conversation-host-and-participant"></a>대화, 호스트 및 참가자 개요

**대화** 는 참여 하는 다른 사용자가 참여 하기 위해 한 사용자가 시작 하는 세션입니다. 모든 클라이언트는 5 자로 된 **대화 코드**를 사용 하 여 대화에 연결 합니다.

각 대화는 다음을 포함 하는 메타 데이터를 만듭니다.
-    대화가 시작 및 종료 된 시간에 대 한 타임 스탬프
-    음성 또는 텍스트 입력에 대 한 각 사용자의 선택 된 애칭 및 주 언어를 포함 하는 대화의 모든 참가자 목록입니다.


대화에는 **호스트** 와 **참가자**라는 두 가지 유형의 사용자가 있습니다.

**호스트** 는 대화를 시작 하 고 해당 대화의 관리자 역할을 하는 사용자입니다.
- 각 대화에는 하나의 호스트만 있을 수 있습니다.
- 대화 기간 동안에는 호스트를 대화에 연결 해야 합니다. 호스트가 대화를 벗어나면 다른 모든 참가자가 대화를 종료 합니다.
- 호스트에는 대화를 관리 하는 몇 가지 추가 컨트롤이 있습니다. 
    - 대화 잠금-추가 참가자의 조인 방지
    - 모든 참가자 음소거-음성 또는 인스턴트 메시지에서 transcribed 여부에 관계 없이 다른 참가자가 대화에 메시지를 보내지 못하도록 합니다.
    - 개별 참가자 음소거
    - 모든 참가자 음소거 해제
    - 개별 참가자 음소거 해제

**참가자** 는 대화를 조인 하는 사용자입니다.
- 참가자는 다른 참가자에 대 한 대화를 종료 하지 않고 언제 든 지 동일한 대화를 그대로 유지 하 고 다시 조인할 수 있습니다.
- 참가자는 대화를 잠그거나 다른 사용자를 음소거/음소거 해제 할 수 없습니다.

> [!NOTE]
> 각 대화에는 최대 100 명의 참가자가 있을 수 있으며, 언제 든 지 10 개의 참가자가 동시에 음성으로 대화할 수 있습니다.

## <a name="language-support"></a>언어 지원

대화를 만들거나 조인할 때 각 사용자가 **기본 언어**를 선택 해야 합니다. 즉, 사용자가 말하는 언어와 인스턴트 메시지를 보낼 언어와 다른 사용자의 메시지가 표시 되는 언어를 선택 해야 합니다.

**음성 텍스트** 와 **텍스트 전용**의 두 가지 언어가 있습니다.
- 사용자가 기본 언어로 **음성-텍스트** 언어를 선택 하는 경우 대화에서 음성 및 텍스트 입력을 모두 사용할 수 있습니다.

- 사용자가 **텍스트 전용** 언어를 선택 하는 경우에는 텍스트 입력을 사용 하 고 대화에서 인스턴트 메시지를 보낼 수만 있습니다. 텍스트 전용 언어는 텍스트 번역에 대해 지원 되지만 음성 텍스트는 지원 되지 않는 언어입니다. [언어 지원](supported-languages.md) 페이지에서 사용 가능한 언어를 볼 수 있습니다.

각 참가자는 주 언어와는 별도로 대화를 번역 하기 위한 추가 언어를 지정할 수도 있습니다.

다음은 사용자가 선택한 주 언어에 따라 다중 장치 대화에서 수행할 수 있는 작업에 대 한 요약입니다.


| 사용자가 대화에서 수행할 수 있는 작업 | 음성 텍스트 변환 | 텍스트 전용 |
|-----------------------------------|----------------|------|
| 음성 입력 사용 | ✔️ | ❌ |
| 인스턴트 메시지 보내기 | ✔️ | ✔️ |
| 대화 변환 | ✔️ | ✔️ |

> [!NOTE]
> 사용 가능한 음성 텍스트 및 텍스트 번역 언어 목록은 [지원 되는 언어](supported-languages.md)를 참조 하세요.
