작업을 완료한 후에는 리소스 그룹을 삭제할 수 있습니다. 리소스 그룹을 삭제하면 저장소 계정, Azure 파일 공유 및 리소스 그룹 내부에 배포한 다른 리소스가 삭제됩니다.

1. 왼쪽 메뉴에서 **리소스 그룹**을 선택합니다.
2. 리소스 그룹을 마우스 오른쪽 단추로 클릭하고 **리소스 그룹 삭제**를 선택합니다. 창이 하나 열리고 리소스 그룹과 함께 삭제될 리소스에 대한 경고가 표시됩니다.
3. 리소스 그룹 이름을 입력하고 **삭제**를 선택합니다.
