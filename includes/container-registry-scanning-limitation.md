> [!NOTE]
> Azure Security Center는 현재 개인 끝점, 선택한 서브넷 또는 IP 주소에 대 한 액세스를 제한 하는 레지스트리에서 [이미지 취약성 검사](../articles/security-center/azure-container-registry-integration.md?toc=/azure/container-registry/toc.json&bc=/azure/container-registry/breadcrumb/toc.json) 를 수행할 수 없습니다. Azure DevOps Services 및 Azure Container Instances를 비롯 한 특정 Azure 서비스의 인스턴스도 네트워크 제한 컨테이너 레지스트리에 액세스할 수 없습니다.
