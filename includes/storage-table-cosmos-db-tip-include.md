> [!TIP]
> 이 문서의 내용은 원래 Azure Table Storage에 적용됩니다. 그러나 이제 테이블 스토리지에 대한 대체 제품인 Azure Cosmos DB Table API가 있습니다. 이 API는 더 높은 성능과 가용성, 글로벌 배포 및 자동 보조 인덱스를 제공합니다. 또한 소비 기반 [서버리스](../articles/cosmos-db/serverless.md) 모드에서도 사용할 수 있습니다. Azure Cosmos DB의 Table API와 Azure 테이블 스토리지 사이에는 몇 가지 [기능 차이](../articles/cosmos-db/table-api-faq.yml)가 있습니다. 자세한 내용은 [Azure Cosmos DB 테이블 API](../articles/cosmos-db/table-introduction.md)를 참조하세요.
