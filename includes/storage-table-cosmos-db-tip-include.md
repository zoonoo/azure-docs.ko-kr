> [!TIP]
> 이 문서의 내용은 원래 Azure Table Storage에 적용됩니다. 그러나 이제 테이블 스토리지에 대한 프리미엄 서비스인 Azure Cosmos DB Table API가 있습니다. 이 API는 처리량 최적화 테이블, 글로벌 배포 및 자동 보조 인덱스를 제공합니다. [Azure Cosmos DB의 Table API와 Azure 테이블 스토리지 간의 기능](../articles/cosmos-db/table-api-faq.md#table-api-vs-table-storage)에는 몇 가지 차이가 있습니다. 프리미엄 환경에 대한 자세한 내용과 사용해 보려면 [Azure Cosmos DB Table API](https://aka.ms/premiumtables)를 참조하세요.
