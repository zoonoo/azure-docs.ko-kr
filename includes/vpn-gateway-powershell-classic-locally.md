클래식 배포 모델을 사용할 경우 Azure Cloud Shell를 사용할 수 없습니다. 대신, 최신 버전의 Azure SM (서비스 관리) PowerShell cmdlet을 컴퓨터에 로컬로 설치 해야 합니다. 이러한 cmdlet은 AzureRM 또는 Az cmdlet과 다릅니다. SM cmdlet을 설치 하려면 [서비스 관리 Cmdlet 설치](/powershell/azure/servicemanagement/install-azure-ps)를 참조 하세요. 일반적인 Azure PowerShell에 대 한 자세한 내용은 [Azure PowerShell 설명서](/powershell/azure/)를 참조 하십시오.
