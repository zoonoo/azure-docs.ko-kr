#### <a name="to-add-or-remove-a-volume"></a>볼륨을 추가 또는 제거하려면

1. StorSimple 디바이스로 이동하고 **백업 정책** 을 클릭합니다.

2. 정책의 테이블 형식 목록에서 수정하려는 정책을 선택 및 클릭합니다. 상황에 맞는 메뉴를 마우스 오른쪽 단추로 클릭하여 호출한 후 **볼륨 추가/제거** 를 선택합니다.

    ![왼쪽 설정 창에 백업 정책이 강조 표시 됩니다. 백업 정책에서 유일한 정책 인 mybupol1가 강조 표시 됩니다. 상황에 맞는 메뉴에서 볼륨 추가/제거가 강조 표시 됩니다.](./media/storsimple-8000-add-remove-volume-backup-policy-u2/addvolbupol1.png)

3. **볼륨 추가/제거** 블레이드에서 해당 확인란을 선택 하거나 선택 취소 하 여 볼륨을 추가 하거나 제거 합니다. 해당 확인란을 선택 하거나 선택 취소 하 여 여러 볼륨을 선택/선택 취소 합니다.

    ![Myssvolsrch2 볼륨이 선택 됩니다. 볼륨 정보는 추가 단추와 마찬가지로 강조 표시 됩니다.](./media/storsimple-8000-add-remove-volume-backup-policy-u2/addvolbupol3.png)

    다른 볼륨 컨테이너에서 백업 정책으로 볼륨을 할당하는 경우 해당 볼륨 컨테이너를 함께 장애 조치해야 함을 기억해야 합니다. 이에 대한 경고 메시지가 표시됩니다.

    ![한 컨테이너의 볼륨이 강조 표시 되 고 다른 컨테이너의 두 볼륨이 함께 강조 표시 됩니다.](./media/storsimple-8000-add-remove-volume-backup-policy-u2/addvolbupol2.png)

4. 백업 정책이 수정되면 알림이 표시됩니다. 백업 정책 목록도 업데이트됩니다.

    ![백업 정책 목록에서 정책 mybupol1의 볼륨 수가 강조 표시 됩니다. 값은 1입니다.](./media/storsimple-8000-add-remove-volume-backup-policy-u2/addvolbupol6.png)
