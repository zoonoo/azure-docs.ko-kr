## <a name="prerequisites"></a>필수 조건

이 자습서를 완료하려면 다음이 설치되어 있어야 합니다.

* **ASP.NET 및 웹 개발 워크로드**가 설치되어 있는 <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017 이상</a>이 포함된 Windows 머신.
* [.NET Core 2.2 SDK](https://dotnet.microsoft.com/download).
* 배포 및 실행할 하나 이상의 디바이스(iOS 또는 Android).
  * Android를 사용하는 경우:
    * Windows 머신에 <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3 이상</a> 및 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a>가 설치되어 있습니다.
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">개발자 사용</a> 및 <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore 지원</a> Android 디바이스.
  * iOS를 사용하는 경우:
    * <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10+</a>, <a href="https://cocoapods.org" target="_blank">CocoaPods</a> 및 <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3+</a>가 설치된 macOS 머신.
    * 개발자가 <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit 호환</a> iOS 디바이스를 활성화했습니다.
