클라이언트에 연결할 파트너 소프트웨어를 구성합니다. 파트너 소프트웨어에 연결하려면 일반적으로 이전 단계에서 로컬 웹 UI의 **연결 및 복사** 페이지에서 수집한 다음 정보(다를 수 있음)가 필요합니다.

- Storage 계정 이름
- 액세스 키
- Blob service 엔드포인트
 
연결이 성공적으로 설정되었는지 확인하려면 Storage 탐색기를 사용하여 외부 스토리지 계정에 연결합니다. Storage 탐색기가 없는 경우 Storage 탐색기를 다운로드합니다.

처음으로 Storage 탐색기를 사용하는 경우 다음 단계를 수행해야 합니다.

1. 맨 위 명령 모음에서 **편집 > 대상 Azure Stack**으로 이동합니다.

    ![Data Box 대시보드](media/data-box-configure-partner-software/data-box-connect-via-rest-6.png)

2. 변경 내용이 적용되도록 Storage 탐색기를 다시 시작합니다.

    ![Data Box 대시보드](media/data-box-configure-partner-software/data-box-connect-via-rest-7.png)
