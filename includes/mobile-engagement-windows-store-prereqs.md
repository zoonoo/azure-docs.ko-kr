
이 자습서를 사용하려면 다음이 필요합니다.

* Visual Studio 2015
* [MicrosoftAzure.MobileEngagement](http://go.microsoft.com/?linkid=9864592) Nuget 패키지

> [!IMPORTANT]
> 이 자습서에는 Visual Studio 15를 사용하는 것으로 가정합니다. Visual Studio 13을 사용할 수도 있지만 사용자 인터페이스가 이 자습서의 인터페이스와 약간 다릅니다.
